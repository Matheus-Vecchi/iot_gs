# Eclipse Protocol - IoT

Protótipo IoT da plataforma de monitoramento agrícola Eclipse Protocol, desenvolvido com ESP32 no simulador Wokwi.

2. Inicie a simulação
3. Ajuste o potenciômetro e o DHT22 para ver o sistema reagindo
